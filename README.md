This project presents a strategic sales and operations dashboard developed in Microsoft Excel, using datasets covering orders, products, regions, and customers.

Key techniques include Pivot Tables for aggregation, XLOOKUP and VLOOKUP for data mapping, and calculated metrics for performance evaluation.

The dashboard analyzes revenue trends, regional performance, product insights, and delivery efficiency. The North region records the highest sales, with clear seasonal peaks in May and November. Delivery performance varies across regions, with an approximate 30% cancellation rate, highlighting opportunities for operational improvement.

Overall, the dashboard supports data-driven decision-making across marketing, logistics, and regional strategy.
